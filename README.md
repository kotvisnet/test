# Демографический мониторинг и прогнозирование населения РФ

Веб-приложение для анализа, визуализации и прогнозирования численности населения России по муниципальным образованиям с элементами ИИ-аналитики.

## Функциональность

### 1. Мониторинг населения
- Численность населения по муниципалитетам
- Динамика изменения населения
- Демографические показатели:
  - рождаемость
  - смертность
  - естественный прирост
  - миграция
- Таблица муниципалитетов с наибольшим ростом/снижением
- Интерактивная карта (выбор через клик)

### 2. Прогнозирование
- Прогноз численности населения на 5–15 лет
- График фактических и прогнозных значений
- Прогнозный интервал
- Метрики качества:
  - MAE
  - RMSE
  - MAPE

### 3. ИИ-аналитика
- Автоматическая генерация аналитической справки
- Выявление тенденций
- Оценка факторов влияния
- Рекомендации по социальной политике

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ВАШ_ЛОГИН/population-ai-app.git
cd population-ai-app

2. Установить зависимости
python -m pip install -r requirements.txt
3. Сгенерировать демо-данные
python generate_demo_data.py
4. Запустить приложение
python -m streamlit run app.py

Открыть в браузере:

http://localhost:8501
